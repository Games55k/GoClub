# GoClub 项目介绍

GoClub 是一个围绕 Go 语言学习、项目实践和技术整理搭建的个人站点。

这个项目使用 Hugo 生成静态页面，通过 GitHub Pages 对外发布，当前主题为 `hugo-book`。整体定位不是传统的时间流博客，而是更适合做文档、笔记和项目记录的知识库型网站。

## 这个项目想做什么

GoClub 主要承载三类内容：

- Go 语言基础与进阶学习笔记
- 个人项目的开发记录与复盘
- 工具使用、部署经验和踩坑总结

随着内容逐步积累，这个站点会从一篇项目介绍开始，慢慢扩展成一个更完整的个人技术站点。

## 当前技术方案

- 静态站点生成器：Hugo
- 部署平台：GitHub Pages
- 页面主题：hugo-book
- 内容格式：Markdown
